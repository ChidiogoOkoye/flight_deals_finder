# ✈️ Flight Deals Finder

A Python application that automatically searches for cheap flight deals and notifies the user when prices drop below a specified threshold.

This project uses APIs to track flight prices and send notifications when a better deal is found.

---

## 🚀 Features

- Searches for flight deals between selected cities
- Compares current flight prices with stored price thresholds
- Automatically detects cheaper flights
- Sends notifications when a deal is found
- Uses external APIs to retrieve real-time flight data

---

## 🛠 Technologies Used

- Python
- Requests
- APIs
- Environment Variables
- Data handling with JSON
